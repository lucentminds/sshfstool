sshfstool is a handy tool kit for use with sshfs.

useage: sshfstool [options] COMMAND

commands:

mount    Mount an existing sshfstool entry.
unmount  Unmount an existing sshfstool entry or all.
add      Add a new mount configuration.
remove   Remove an existing mount configuration.

options:
   -h    --help      print help
